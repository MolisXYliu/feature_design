# Code Review Helper Plugin

一个用于代码审查和 Git 摘要生成的测试插件。

## 包含组件

### Skills
- **code-review** — 对指定文件进行代码审查，输出分级的审查报告
- **git-summary** — 分析 Git 提交历史，生成智能变更摘要

### MCP Servers
- **demo-time-server** — 示例 MCP 服务器（仅用于测试，URL 不可用）

## 安装方式
1. 在 CMBDevClaw 中进入 自定义 → Plugins
2. 点击 "+" 按钮
3. 选择此文件夹 或 上传 .zip 文件
